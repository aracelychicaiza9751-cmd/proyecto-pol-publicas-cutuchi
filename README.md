# Evaluación de la Política de Saneamiento Ambiental en la Cuenca del Río Cutuchi

**Autora:** Aracely Chicaiza  
**Institución:** Universidad Técnica de Cotopaxi (UTC)  
**Asignatura:** Políticas Públicas  

---

## Descripción del Proyecto
Este proyecto evalúa los impactos de los vertidos de aguas servidas e industriales en la cuenca del Río Cutuchi (Latacunga - Ecuador) mediante una arquitectura multiagéntica de simulación e Inteligencia Artificial.

---

## Estructura del Repositorio
- `data/processed/`: Base de datos procesada en formato CSV.
- `simulation/`: Código fuente en Python con el modelo multiagente (`Mesa`).
- `docs/`: Documentación de la arquitectura de agentes de IA y bitácora de gobernanza.
