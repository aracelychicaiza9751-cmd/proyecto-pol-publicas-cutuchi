Dashboard Analítico: Política Pública Cutuchi

Evaluación de impacto econométrico, simulación ambiental y análisis de propensión para la cuenca del río Cutuchi.

Distribución del Propensity Score

Gráfico de densidad construido a partir del puntaje de propensión estimado con variables observables previo al tratamiento. El soporte común identifica el rango de puntajes donde conviven tratados y controles.

Comparación de Ingresos

Comparación de ingresos promedio antes y después del emparejamiento. Este gráfico ilustra la reducción de brechas y el contrafactual construido con nearest neighbor matching.

Notas metodológicas

El cálculo del Propensity Score se realiza con una especificación de regresión logística utilizando covariables previas al tratamiento. El emparejamiento se define según nearest neighbor para construir un contrafactual de control comparable dentro de la cuenca.

Fuentes de datos y procesamiento econométrico

Los gráficos cargan desde scripts de Python y calculan el soporte común y la comparación de ingresos con datos empíricos emparejados. El dashboard procesa indicadores clave orientados a la gestión de políticas públicas ambientales.
